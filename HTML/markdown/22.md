# JavaScript 常用运算符


## JavaScript 算数运算符
算数运算符用于对数字执行算数运算：

|运算符	|描述|
|--|--|
|+	|加法|
|-	|减法|
|*	|乘法|
|/|	除法|
|%	|系数|
|++	|递加|
|--	|递减|

## JavaScript 赋值运算符

```
x+=8
x=x+8
```

## JavaScript 比较和逻辑运算符
比较和逻辑运算符返回 true 或 false。  

### 比較運算符

|运算符	|描述	|
|--|--|
|==	|等于	|
|===	|值相等并且类型相等|
|!=	|不相等	|
|!==	|值不相等或类型不相等	|
|>	|大于	|
|<	|小于	|
|>=	|大于或等于|
|<=	|小于或等于|

### 邏輯運算符
逻辑运算符
逻辑运算符用于判定变量或值之间的逻辑。  

```
&&	与	
||	或	
!	非	
```
